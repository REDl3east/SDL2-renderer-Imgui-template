# SDL2 Renderer & Imgui Template

## Quick Start
```bash
mkdir build
cd build
cmake .. # It may take a little bit!
make -j 8
```

## Controls

[**q**]&nbsp;&nbsp;&nbsp;- quits<br>
[**F11**]&nbsp;- toggle fullscreen


## Dependancies
- cmake 3.14+
- SDL2
- Imgui

Cmake will take care of SDL2 and Imgui, so you really only need the correct cmake version!
